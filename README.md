# Student Grade Management System

## By Darshil Jha

---

## Project Overview

This is a simple code made in Python to manage student grades. It allows users to perform basic operations like Create, Read, Update, Delete on student records, view  statistics, and visualize student marks using various types of graphs.

## Features

*   **Add/Update Student:** Add new students to the system or update marks for existing students.
*   **Delete Student:** Remove a student's record from the system.
*   **View All Students:** Display a list of all students along with their respective marks.
*   **Grade Analytics:**
    *   View the highest marks achieved by a student (Topper marks).
    *   Calculate and display the average marks of all students.
*   **Interactive Visualization:** Generate and display different types of graphs for student marks:
    *   Bar Graph
    *   Pie Chart
    *   Scatter Plot
